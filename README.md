# Analítica Retail Óptica con Power BI

## Descripción

Proyecto de analítica de datos enfocado en una tienda de óptica con múltiples sucursales. El objetivo es diseñar un modelo dimensional (Data Warehouse) que permita analizar ventas, productos, clientes y sucursales para apoyar la toma de decisiones.

## Objetivos

* Construir un modelo de datos orientado a negocio
* Simular datasets basados en un catálogo real de productos
* Desarrollar dashboards en Power BI
* Definir KPIs clave del negocio retail óptico

## Estructura del proyecto

* `data/`: datos crudos y procesados
* `notebooks/`: exploración y análisis
* `scripts/`: procesos ETL
* `powerbi/`: archivo del dashboard
* `docs/`: documentación del modelo y diccionario de datos
* `assets/`: imágenes y recursos
* `config/`: configuraciones del proyecto

## Tecnologías

* Python (Pandas)
* SQL
* Power BI
